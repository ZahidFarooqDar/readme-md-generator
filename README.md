# Welcome to readme-md-generator 👋
CLI that generates beautiful README.md files
## 🏠  [Homepage](github.com/ZahidFarooqDar/readme-generator)
## Prerequisities
* npm >=5.5.0
* node >=9.3.0
## Install
```sh
npm install
```
## Usage
```sh
npm run start
```
## Run tests
```sh
npm run test
```
